Welcome to Abhaysai's Text Summarizer

Purpose & Overview:

The purpose of this project was to familiarize myself with utilizing python's NLTK library, understand and develop API's, and learn how to communicate between client-side and server-side. In this project users will be able to input a piece of text and recieve an extractive summary with the amount of sentences they requested: Source Code.

Pitfalls:

The summarization method used is extractive summarization. This entails generating a rank for each sentence and returning the top n ranked sentences. Pitfalls of this include some loss of information as the summarization method is not abstractive.
